# Reglas de Negocio

## Comercial

- Todo pedido debe tener un cliente.
- Todo cliente debe estar registrado.
- Un pedido puede generar uno o varios servicios.

## Operaciones

- Todo servicio debe tener un responsable.
- Todo servicio debe tener un estado.
- Todo servicio debe quedar asociado a un pedido.

## Legalizaciones

- No puede cerrarse un servicio sin legalización.
- Todo gasto debe asociarse a un servicio.

## Financiero

- No puede generarse factura sin servicio entregado.
- Toda factura debe asociarse a un cliente.
- Todo pago debe asociarse a una factura.

## Control

- Todas las acciones deben quedar auditadas.
